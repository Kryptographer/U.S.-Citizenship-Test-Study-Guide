A free, interactive study tool to help you prepare for the USCIS Naturalization Civics Test. This application contains all 100 official civics questions with enhanced learning features including mnemonics, explanations, practice games, and progress tracking.

## Getting Started

### Option 1: Open Directly in Your Browser

1. Download this folder to your computer
2. Find the file called `index.html`
3. Double-click on it - it will open in your web browser
4. Start studying immediately

### Option 2: Install as an App (PWA)

Modern browsers allow you to install this as a standalone app:
1. Open `index.html` in Chrome, Edge, or Safari
2. Look for the "Install" option in your browser menu
3. The app will work offline once installed

### Option 3: Use Online

If hosted online, simply visit the provided website link.

## Study Modes

### Flash Cards

- Flip through all 100 questions one at a time
- Click or tap a card to reveal the answer
- **Keyboard shortcuts:**
  - Arrow keys (← →) - Navigate between cards
  - Spacebar - Flip the card
  - M - Mark for later review
- View explanations and memory aids for each question

### Practice Quiz

- Choose question count: 10, 20, 50, or all 100
- Filter by topic or category
- Three answer modes:
  - **Written**: Type your answer for better memorization
  - **Multiple Choice**: Pick from 4 options
  - **Reveal**: Self-grade mode

### Practice Exam

Simulates the real citizenship test experience:
- **Standard Mode**: Answer all 10 questions
- **Realistic Mode**: Pass when you get 6 right (like the real test)
- **Timed Mode**: 10-minute time limit

### Study Guide

- Browse all 100 questions organized by topic
- Search for specific topics
- Filter by category
- View detailed explanations for each question

### Interactive Games

- **Speed Challenge** - Answer as many questions as possible in 60 seconds

- **Fill in the Blank** - Complete sentences with missing words
- **Daily Challenge** - New challenge every day with streak tracking

### Gamification

- Earn points for correct answers
- Level up as you progress
- Unlock achievements
- Track your study streaks

## Special Features

### 65/20 Rule

If you are 65 years or older AND have been a permanent resident for 20+ years, you only need to study 20 specific questions. Use the "65/20 Questions Only" filter to focus on just those questions.

### Mark for Review

Click the bookmark icon to mark difficult questions. Filter to show only marked questions later.

### Works Offline

Once loaded, the application works without internet. Perfect for studying anywhere.

### Saves Your Progress

Your study progress, preferences, and achievements are saved automatically in your browser.

## Frequently Asked Questions

**Q: Do I need to install anything?**
A: No. Just open the `index.html` file in any web browser. Optionally, you can install it as a PWA for a native app experience.

**Q: Does this work on my phone or tablet?**
A: Yes. The application is fully responsive and works on all devices.

**Q: Will my progress be saved?**
A: Yes. Progress is saved in your browser. Clearing browser data or using a different browser/device will reset progress.

**Q: Is this the official USCIS test?**
A: No. This is an unofficial study tool. See the disclaimer below.

**Q: How do I reset my progress?**
A: Go to the Progress section and click "Reset Progress."

**Q: How do I enable dark mode?**
A: Click the moon icon in the navigation to toggle dark mode.
